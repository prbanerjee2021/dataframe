Introduction to Python Libraries

Pandas

Pandas is a powerful open-source data analysis and manipulation library for Python. It provides data structures and operations for manipulating numerical tables and time series.

Key features:

  • Data structures: Series (1D) and DataFrame (2D)
  • Data alignment and handling of missing data
  • Reshaping and pivoting of datasets
  • Label-based slicing, indexing, and subsetting
  • Robust input/output tools for various file formats

Matplotlib

Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python. It produces publication-quality figures in a variety of formats.

Key features:

  • Line plots, bar charts, histograms, scatter plots, etc.
  • Customization of plot styles (labels, colors, markers, etc.)
  • Support for LaTeX formatted text and symbols
  • Interactive features like zooming, panning, and updating
  • Integration with Pandas DataFrames

Data Structures in Pandas

Pandas has two primary data structures:

Series

A one-dimensional labeled array capable of holding any data type.

import pandas as pd

# Creating a Series
s = pd.Series([1, 3, 5, 7, 9])
print(s)
DataFrame

A two-dimensional labeled data structure with columns that can be of different types.

# Creating a DataFrame
data = {'Name': ['Alice', 'Bob', 'Charlie'],
        'Age': [25, 30, 35],
        'City': ['New York', 'Paris', 'London']}

df = pd.DataFrame(data)
print(df)

Pandas Series

Creating Series

From ndarray
import numpy as np
import pandas as pd

# Create from NumPy array
arr = np.array([10, 20, 30, 40])
s1 = pd.Series(arr)
print(s1)
From Dictionary
# Create from dictionary
data = {'a': 10, 'b': 20, 'c': 30}
s2 = pd.Series(data)

From Scalar Value
# Create with scalar value
s3 = pd.Series(5, index=[0, 1, 2, 3])

Mathematical Operations

Series support vectorized operations:

s = pd.Series([1, 2, 3, 4, 5])

# Basic operations
print("Addition:", s + 10)
print("Multiplication:", s * 2)
print("Exponentiation:", s ** 2)

# Operations between Series
s2 = pd.Series([10, 20, 30, 40, 50])
print("Series Addition:", s + s2)
print("Series Multiplication:", s * s2)

Head() and Tail() Functions

These functions allow you to view the top or bottom rows of a Series:

s = pd.Series(range(1, 101))  # Series from 1 to 100

# View first 5 elements
print(s.head())

# View first 10 elements
print(s.head(10))

# View last 5 elements
print(s.tail())

# View last 10 elements
print(s.tail(10))

Selection, Indexing and Slicing

Accessing data in a Series:

s = pd.Series([10, 20, 30, 40, 50], index=['a', 'b', 'c', 'd', 'e'])

# By index label
print("Element at 'b':", s['b'])

# By position
print("Element at position 2:", s[2])

# Slicing by index labels
print("Elements from 'b' to 'd':\n", s['b':'d'])

# Slicing by position
print("Elements from position 1 to 3:\n", s[1:4])

# Conditional selection
print("Elements greater than 30:\n", s[s > 30])

Operations on Rows and Columns

Adding Columns

Selecting Columns

Renaming Columns

Indexing

Label-based Indexing (loc)
# Set index
df.set_index('Full Name', inplace=True)

# Access row by label

# Access specific values

Boolean Indexing
# Filter rows based on condition

# Multiple conditions

Importing/Exporting CSV Files

Reading from CSV Files

# Read CSV file
df = pd.read_csv('data.csv')

# Read with specific options
df = pd.read_csv('data.csv', 
                 header=0,        # Use first row as headers
                 index_col='id',   # Set 'id' column as index
                 usecols=['id', 'name', 'age'],  # Select specific columns
                 na_values=['NA', 'missing'])    # Treat these as NaN

Writing to CSV Files

# Write to CSV
df.to_csv('output.csv')

# Write with specific options
df.to_csv('output.csv', 
          index=False,        # Don't write index
          columns=['name', 'age'],  # Write specific columns
          na_rep='NULL')      # Represent NaN as NULL

Data Cleaning Example

# Read data
df = pd.read_csv('student_data.csv')

# Handle missing values
df['Math'].fillna(df['Math'].mean(), inplace=True)
df['Science'].fillna(df['Science'].median(), inplace=True)

# Convert data types
df['Grade'] = df['Grade'].astype('category')

# Save cleaned data
df.to_csv('cleaned_student_data.csv', index=False)

Data visualization helps in:

Data Science Glossary

Essential Terms

DataFrame

A 2-dimensional labeled data structure with columns of potentially different types, similar to a spreadsheet or SQL table.

Series

A one-dimensional labeled array capable of holding any data type (integers, strings, floating point numbers, Python objects, etc.).

NaN (Not a Number)

A special floating-point value used to represent missing or undefined data in Pandas.

Indexing

The process of selecting specific rows and columns of data in a DataFrame using labels (loc) or integer positions (iloc).

Boolean Indexing

A method of selecting data from a DataFrame using conditions that return boolean values (True/False).

CSV (Comma-Separated Values)

A plain text file format used to store tabular data, where each line represents a data record and each record consists of fields separated by commas.

Matplotlib

A comprehensive library for creating static, animated, and interactive visualizations in Python.

Histogram

A graphical representation of the distribution of numerical data, where the data is divided into bins and the count of observations in each bin is shown.

Explain `iloc` Statements

Explain what the following statements are doing? df is the name of the dataframe.

a) df.iloc[:5,]

This selects the first 5 rows (from the beginning up to, but not including, index 5) and all columns of the DataFrame df. The iloc attribute stands for “integer location” and is used to access a DataFrame using integer positions. The trailing comma is optional.

b) df.iloc[1:5]

This retrieves rows with integer positions from 1 up to, but not including, 5. This means it selects rows at index 1, 2, 3, and 4. All columns for these rows are returned.

c) df.iloc[5,0]

This selects the specific value at the 6th row (index 5) and the 1st column (index 0) of the DataFrame. It returns a single scalar value.

Note: A slice like df.iloc[5:0] would result in an empty slice, as the start index (5) is greater than the stop index (0).
d) df.iloc[1:5,0]

This selects rows from index 1 to 4 (1:5) and only the first column (index 0). It returns a Pandas Series containing the values from the first column for the specified rows.

e) df.iloc[1:5,:5]

This selects a subset of the DataFrame. It gets rows from index 1 to 4 (1:5) and columns from index 0 to 4 (:5).

f) df.iloc[2:7,1:3]

This selects rows from index 2 to 6 (2:7) and columns from index 1 to 2 (1:3). It returns a new DataFrame that is a slice of the original.

.loc vs .iloc Slicing

Trying to extract the first five rows of DataFrame x, Aman has given code as: x.loc[0:5]. But it is returning 6 rows. Why? Suggest the solution.

Reason: The .loc indexer is label-based, and when slicing with labels (even if they are default integer labels), it includes both the start and the end index. So, x.loc[0:5] selects all rows from label 0 to label 5, inclusive, which results in 6 rows (0, 1, 2, 3, 4, 5).

Solution: To get the first 5 rows, you can use one of the following methods:

  • Using .loc with the correct end label: x.loc[0:4] (This will select rows with labels 0 through 4).
  • Using .iloc for integer-position slicing: x.iloc[0:5] (.iloc is exclusive of the end index, so it will select rows at positions 0, 1, 2, 3, and 4).
  • Using the .head() method: x.head(5) (This is often the most readable way to get the first n rows).

DataFrame Operations

Given a DataFrame namely “employee” that stores the details of employees. First, let's create and view the DataFrame.

import pandas as pd
data = {'Name': ['Aman', 'Riya', 'Darpan', 'Sonia', 'Karan'],
        'Designation': ['Manager', 'Analyst', 'Sr. Analyst', 'Developer', 'HR'],
        'Salary': [120000, 85000, 95000, 90000, 75000],
        'Bonus': [15000, 8000, 10000, 9500, 7000]}
employee = pd.DataFrame(data, index=['E101', 'E102', 'E103', 'E104', 'E105'])
print(employee)
# Output:
#          Name  Designation  Salary  Bonus
# E101    Aman      Manager  120000  15000
# E102    Riya      Analyst   85000   8000
# E103  Darpan  Sr. Analyst   95000  10000
# E104   Sonia    Developer   90000   9500
# E105   Karan           HR   75000   7000

Now, write statements to perform the following tasks:

a) Write a statement to display Name, Designation and Salary columns.
print(employee[['Name', 'Designation', 'Salary']])
b) Write a statement to display Name and Salary columns.
print(employee[['Name', 'Salary']])
c) Write a statement to display Bonus column only.
# This returns a Series
print(employee['Bonus'])

# To return a DataFrame, use double square brackets
print(employee[['Bonus']])
d) Write a statement to display all the information from Employee ids ‘E102’ to ‘E104’ (Both are included).

Since the index is label-based, we use .loc which is inclusive of the end label.

print(employee.loc['E102':'E104'])
e) Write a statement to display all information of Employee id ‘E102’ and ‘E105’.

We pass a list of labels to .loc to select non-consecutive rows.

print(employee.loc[['E102', 'E105']])
f) Write a statement to display the employee’s name, designation and salary for those having employee ids as ‘E101’ and ‘E103’.
print(employee.loc[['E101', 'E103'], ['Name', 'Designation', 'Salary']])
g) Write a statement to display the employee’s name and bonus for those having employee id as ‘E101’ and ‘E103’.
print(employee.loc[['E101', 'E103'], ['Name', 'Bonus']])
h) Write a statement to display the Salary of Darpan.

Darpan's Employee ID is 'E103'.

# Using .loc with row and column labels
print(employee.loc['E103', 'Salary'])

# Alternative using boolean indexing (if index was not set)
# print(employee[employee['Name'] == 'Darpan']['Salary'])
